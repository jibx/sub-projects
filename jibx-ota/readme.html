Welcome to the 1.0 Release

This is the first public release of the JiBX/OTA subproject, providing JiBX support for working with the Open Travel Alliance schemas widely used within the travel industry. The subproject includes two different variations of code generation for the OTA schemas using the standard JiBX CodeGen tool, a WSDL generation example using the Jibx2Wsdl tool, and two sample web service client and server implementations, the first using Axis2 with JiBX data binding and the second using the JiBX/WS web services subproject.

The docs subdirectory contains the full JiBX/OTA documentation, which is also available online. See the documentation for more information about the project.
